# ElClimaQuintoDePrimaria
Herramienta pedagógica diseñada para trabajar los contenidos del bloque de climatología con alumnado de quinto de Educación Primaria. Se presenta como un recurso de aprendizaje gamificado que combina la interacción con la pantalla con actividades de lectura, escritura y reflexión en soporte físico y que reserva al docente un papel central y activo.
